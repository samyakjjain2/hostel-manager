# Aegis | Hostel Operations System (React + Node + PostgreSQL Rebuild)

A complete production-grade Hostel Management System designed for the **Hostel Manager/Admin**. Built with React (Vite + Tailwind CSS v4) and Node.js (Express + Prisma ORM + PostgreSQL).

---

## 📂 Project Architecture

```
hostel-management-app/
├── frontend/                # React client application (Vite, Tailwind, React Router)
├── backend/                 # Node.js API server (Express, Prisma, PostgreSQL)
└── backup_vanilla/          # Backup of the original HTML/JS static mock application
```

---

## ⚡ Quick Start Instructions

Follow these steps to set up and run the system locally:

### Step 1: Configure the Database
1. Open the file `backend/.env`.
2. Replace `DATABASE_URL` with your actual **PostgreSQL** database connection string (e.g., Neon DB).
   ```env
   DATABASE_URL="postgresql://user:password@your-neon-host/hosteldb?sslmode=require"
   ```

### Step 2: Set Up Backend Database Tables
Navigate to the `backend/` folder and push the Prisma schema to create all tables, indices, and constraints:
```bash
cd backend
npm run db:push
```

### Step 3: Seed Dummy Datasets
Seed the database with default hostels, rooms, notices, staff members, and a manager login:
```bash
npm run db:seed
```

### Step 4: Run the Backend API Server
Start the Express server on port `5000`:
```bash
npm run dev
```

### Step 5: Run the Frontend Client Dev Server
Open a new terminal window, navigate to the `frontend/` folder, install packages and start the Vite dev server:
```bash
cd frontend
npm run dev
```

## 🛠️ Features Included

1. **Dashboard Analytics:** Charts showing occupancy rates, monthly fee collection flow, audit logs, and recent student admissions.
2. **Hostel Management:** Create, edit, and delete wings, including floors, configurations, and rule listings.
3. **Room Inventory:** Floor plans, capacities, AC status, bathroom setups, and custom pricing.
4. **Student Directory:** Search/filters for student profiles, emergency contacts, college info, and check-in history.
5. **Room Allocations:** One-click check-in, checkout, and room transfer operations.
6. **Fee Management:** Auto-generate monthly invoices for all room occupants, collect payments, and track outstanding fees.
7. **Notice Board:** Pin alerts, edit rules, and broadcast announcements.
8. **Complaints Panel:** Kanban task tracker for maintenance tickets.
9. **Visitor Log:** Register parent check-ins and log exits.
10. **Staff Directory:** Manage department roles, salaries, and phone rosters.
